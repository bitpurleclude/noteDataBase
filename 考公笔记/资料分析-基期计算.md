# 行测做题技巧 - 资料分析

## 基期计算方法

### 1. 直接代入法
- 适用条件：选项简单或增长率较小时
- 公式：$基期 = \frac{现期}{1 + 增长率}$

### 2. 假设做差法
适用条件：增量 < 5%时

| 类型 | 基期估算       | 增量处理     | 现期验证     |
|------|----------------|--------------|--------------|
| 真实 | 368-201        | 5%增长率     | 36883        |
| 假设 | 400            | 20增量       | 420          |
### 3. 假设缩减法
| 参数  | 基期假设 | 增量假设 | 现期结果 | 误差修正           |
| --- | ---- | ---- | ---- | -------------- |
| 初始  | 7    | 1.54 | 8.54 | 5/6*0.14 ≈0.12 |
| 实际  |      | 22%  | 8.4  |                |
